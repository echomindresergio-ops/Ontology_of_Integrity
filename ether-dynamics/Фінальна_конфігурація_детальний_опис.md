# Фінальна стабільна конфігурація ефірної моделі галактики
## Докладний технічний опис для продовження дослідження

*Цей документ описує єдину конфігурацію моделі, яка після довгого циклу діагностики одночасно задовольнила всі встановлені критерії: не розганяється (немає каскадної нестабільності), зберігає прийнятний енергетичний бюджет на космологічному часі (13.8 млрд років), відтворює периферійне домінування активності, і за додаткової умови (вітер) дає нетривіальну, немонотонну форму DM(b).*

---

## 1. Фізична картина

Галактика описується як радіально-симетрична система з трьома полями, визначеними на кожному радіусі r:

- **u_b(r,t)** — баріонна енергетична густина (нормована, безрозмірна; u_b(r,0) задає початковий баріонний профіль).
- **u_e(r,t)** — "ефірна" енергетична густина, що виникає локально з баріонної через поріг напруги, і переноситься в просторі адвекцією та дифузією.
- **Ψ(r,t)** — локальна накопичена "напруга" в кожній комірці; при досягненні порогу відносно u_e/u_b відбувається дискретна подія: миттєва втрата енергії, часткова перекачка ефіру назад у баріони, і скидання Ψ.

Модель НЕ використовує безперервний зворотний потік конденсації (параметр γ був перевірений і виявився фактично нульовим/зайвим у поточній архітектурі — залишений вимкненим).

## 2. Просторова сітка

| Параметр | Значення |
|---|---|
| Тип сітки | Логарифмічна за радіусом |
| r_min | 0.1 пк |
| r_max | 3 Мпк |
| N (робоча, підтверджена до 13.8 Gyr) | 128 |
| N (тестовано частково, обчислювально дороге) | 256 |

Дивергенція потоку обчислюється у скінченно-об'ємній формі на межах комірок (геометричне середнє сусідніх центрів), з множником r² (сферична симетрія).

---

## 3. Початковий баріонний профіль

```
ρ_b(r) = 1 / (1 + sqrt(r / r_s))²,   r_s = 10 кпк
```

Нормовано так, що ρ_b(0) = 1 (центральне значення). Це псевдоізотермічний, "cored" профіль (не NFW-подібний куспід), обраний як представницький, слабко концентрований випадок — саме такий профіль дав найбільш переконливе периферійне домінування в перевірках стійкості.

**Початкові умови:**
```
u_b(r, 0) = ρ_b(r)
u_e(r, 0) = 1e-6 · ρ_b(r)      (малий, але не нульовий початковий ефір)
Ψ(r, 0)   = 0   для всіх r
```

---

## 4. Рівняння еволюції

### 4.1 Джерело (локальна конверсія баріон → ефір)

```
S(r,t) = κ_flow · f(Ψ(r,t)) · [u_b(r,t) / u_b0_center]

f(Ψ) = max(Ψ/Ψ_c, 0)^m
```

### 4.2 Транспорт ефіру (адвекція + дифузія)

```
∂u_e/∂t + (1/r²) ∂(r² J_r)/∂r = S(r,t)
J_r(r,t) = u_e(r,t)·v_e(r) − D·∂u_e/∂r

v_e(r) = v_0·(r/r_0)^(−1/2) + v_wind         (базова "ротаційна" складова + вітер)
```

### 4.3 Еволюція баріонів

```
∂u_b/∂t = −S(r,t)
```

### 4.4 Накопичення напруги

```
∂Ψ/∂t = Θ_0     (стала швидкість накопичення; НЕ включає безперервний "витік")
```

### 4.5 Дискретна локальна подія (у комірці i, коли озброєна)

Умова спрацювання: `u_e[i]/u_b[i] ≥ β_on` (і комірка "озброєна" — див. гістерезис нижче).

При спрацюванні, миттєво:
```
tot = u_b[i] + u_e[i]
loss = L_c · tot
u_b[i] -= loss · (u_b[i]/tot)
u_e[i] -= loss · (u_e[i]/tot)

u_b[i] += ξ · u_e[i]      # перекачка
u_e[i] -= ξ · u_e[i]... тобто u_e[i] *= (1-ξ) після перекачки

Ψ[i] *= (1 − κ_reset)     # часткове скидання напруги
```
Комірка "роззброюється" (не може знову спрацювати) доти, доки `u_e[i]/u_b[i]` не впаде нижче `β_off(r_i)`.

### 4.6 Радіально залежний гістерезис (КЛЮЧОВИЙ елемент стабільності)

Це найважливіше архітектурне рішення, знайдене під час діагностики — єдиний спосіб отримати одночасно стабільність і периферійне домінування:

```
f(r) = 1 − 1/(1 + r/r_s,h)
β_off(r) = β_c + (β_p − β_c) · f(r)

β_c = 0.70   (жорсткий гістерезис у центрі — запобігає каскаду)
β_p = 0.98   (м'який гістерезис на периферії — дозволяє легке повторне озброєння)
r_s,h = 30 кпк
```

## 5. Повна таблиця параметрів фінальної конфігурації

| Параметр | Значення | Роль |
|---|---|---|
| κ_flow | 1×10⁻¹⁶ с⁻¹ | Інтенсивність прямого потоку баріон→ефір |
| Θ₀ | 1×10⁻¹⁵ с⁻¹ | Швидкість накопичення напруги Ψ |
| **L_c** | **3.0×10⁻⁵** | **Втрата енергії за одну локальну подію (виправлене значення!)** |
| ξ | 0.1 | Частка миттєвої перекачки ефір→баріон при події |
| κ_reset | 0.9 | Частка скидання Ψ при події |
| β_on | 1.0 | Поріг активації (глобальний, однаковий скрізь) |
| β_c | 0.70 | Поріг деактивації в центрі (радіальний гістерезис) |
| β_p | 0.98 | Поріг деактивації на периферії (радіальний гістерезис) |
| r_s,h | 30 кпк | Масштаб переходу радіального гістерезису |
| m (показник f(Ψ)) | 1 | — |
| Ψ_c | 1.0 | Нормувальний масштаб напруги |
| D | 1×10²⁵ м²/с | Коефіцієнт дифузії ефіру |
| v₀ | 200 км/с | Амплітуда базової адвекційної швидкості |
| r₀ | 1 кпк | Масштаб базової швидкості |
| **v_wind** | **300 км/с** (стала) | **Додаткова зовнішня адвекція — дає потрібну форму DM(b)** |
| r_s (баріонний профіль) | 10 кпк | Масштаб початкового ρ_b(r) |
| r₂₀₀ | 250 кпк | Умовний віріальний радіус для DM(b)-інтеграції |

**КРИТИЧНО:** L_c=3.0×10⁻⁵ — це виправлене значення. Початкове L_c=0.01 (1%) давало катастрофічне виснаження енергії за ~4.2 млрд років. Значення підібране так, щоб при реальній підтвердженій частоті подій (~900/Gyr) сумарна втрата за 13.8 млрд років (~12420 подій) залишала ≥95% енергії.

## 7. Відомі обмеження цієї конфігурації (чесно, для наступних дослідників)

1. **N=256 не підтверджено на повному часі** — обчислювально занадто дороге (густий якобіан BDF-розв'язувача SciPy). Для справжнього підтвердження на вищій роздільності потрібна оптимізація (розріджений якобіан, Numba/JIT, або сторонній розв'язувач на кшталт SUNDIALS/PETSc).
2. **Переклад u_e → реальна густина електронів (n_e)** — досі довільний, лінійний, без фізичного обґрунтування. Будь-яке кількісне порівняння з реальними DM-даними впирається в цей нерозв'язаний крок.
3. **γ (безперервна конденсація) вимкнена** — перевірена окремо і виявилась або "мертвим" параметром, або другорядним модифікатором, що ніколи не запобігає дискретній події самостійно.
4. **Профіль вітру** — перевірено лише сталий (constant) і два прості альтернативні профілі (peaked, power-law); сталий дає найсильніший, найкраще перевірений на даних ефект, але не обов'язково найфізичніший.
5. **Порівняння з реальними даними DM_excess(b) все ще попереднє** — точні оцифровані таблиці (Hussaini et al. 2025, Рис. 6) не отримані; використано лише наближені орієнтири й окремо перевірений, виправлений параметр концентрації mNFW (c=7.67) з реального коду пакету FRB.
