# Ефірна модель гравітації — дослідницький репозиторій

Незалежне дослідження альтернативної моделі темної матерії/гравітації
("ефір" — структурована модель вакууму), від філософської концепції до
працюючої, стабілізованої комп'ютерної симуляції та першого зіткнення з
реальними астрономічними даними (дисперсія швидких радіосплесків, FRB).

**Статус:** відкрите дослідження. Ключовий кількісний результат —
попередній, заснований на малій вибірці (25 унікальних FRB), і потребує
підтвердження на більших даних. Усі знайдені обмеження й негативні
результати задокументовані поруч із позитивними — див.
`README_статус_дослідження.md` для повного, збалансованого звіту.

## Встановлення

```bash
pip install numpy pandas scipy astropy h5py requests
```

## Швидкий старт

**Відтворити фінальну стабільну модель галактики:**
```bash
python3 ether_galaxy_model.py
```

**Відтворити головний кількісний результат (порівняння моделей проти реальних даних):**
```bash
python3 analysis_model_comparison.py
```

**Провести власний крос-матчинг FRB × Legacy Survey DR9** (потрібен мережевий доступ до `legacysurvey.org` — див. `Інструкція_крос_матчинг_FRB.md`, включно з обов'язковим діагностичним Кроком 0):
```bash
python3 frb_legacysurvey_crossmatch.py
```

## Головний знайдений результат (коротко)

На реальних, крос-матчених даних (72 пари FRB–галактика в межах гало, 25
унікальних FRB) стандартна модель густини гало (модифікований профіль
NFW, фідуціальна концентрація c=7.67) систематично гірше узгоджується з
даними (χ²/dof = 3.0–5.6), ніж альтернативні моделі (ефірна модель:
χ²/dof = 1.0–3.6; тривіальна плоска модель: χ²/dof = 0.5–3.3). Результат
відтворено на двох незалежних схемах бінування.

**Це попередній, а не остаточний результат.** Ефірна модель поки не
перевершує статистично тривіальну "плоску" альтернативу — лише стандартна
модель відстає від обох. Див. `README_статус_дослідження.md` для повного,
чесного розбору значущості й обмежень.

статей і, за потреби, зв'яжіться з авторами (лист-запит уже підготовлено
в `Лист_запит_Hussaini.md`).

## Як продовжити дослідження

Дивіться розділ "Наступні кроки" в `README_статус_дослідження.md` і повне
технічне завдання в `Завдання_крос_матчинг_FRB_гало.md`. Коротко:
1. Накопичення більшої статистики через крос-матчинг (конвеєр готовий).
2. Відповідь від авторів Hussaini et al. 2025 з офіційними даними.
3. Оптимізація коду моделі для підтвердження на вищій просторовій
   роздільності (N≥256) на повному космологічному часі.
